# Personal Portfolio 🌐

Professional portfolio website to showcase projects & skills.  
Built with **React.js + Sanity CMS** for dynamic content management.

## Features
- Dynamic project showcase
- Sanity CMS integration
- Responsive UI
- Easy content updates

## Tech Stack
- Frontend: React.js
- CMS: Sanity
- Styling: TailwindCSS
